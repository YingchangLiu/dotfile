# kelen's dotfiles

**Warning**: Don’t blindly use my settings unless you know what that entails. Use at your own risk!

I'm working with Arch Linux and Debian.

## Installing

```console
$ sh ./install.sh
```
This will create symlinks from this repo to your home folder.
